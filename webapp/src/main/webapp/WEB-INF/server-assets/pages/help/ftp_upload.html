Send data files to ArrayExpress by FTP

ArrayExpress provides a FTP repository accessible by all submitters to upload a large volume of data files which are associated with their experiment submissions. Raw sequence files from high-throughput sequencing experiments (e.g. fastq.gz or bam files) must be transferred to ArrayExpress via FTP as they are too large to be uploaded via the MAGE-TAB or Annotare submission tools.

FTP upload is also useful if you would like to send files to us prior to a submission (e.g. for us to check if they are in a suitable format).

Before you start transferring files, please note the following points:

  1. Make sure your file names are unique/specific so you will not be overwriting other submitters' files (or risk your own files being overwritten likewise). E.g. "Bloggs_lab_Mar2014_sample1.fq.gz" is a better choice of file name than simply "sample1.fq.gz".
  2. When the transfer is complete, please email us at arrayexpress@ebi.ac.uk with the names of files transferred.
  3. All ArrayExpress submitters use the same account to upload files (see below). As most of the files are un-/pre-published, we make all uploaded files "private", which means you will not be able to see any files or directories already on the FTP site, including those that you uploaded or created.
  4. Data files uploaded to ArrayExpress by FTP only constitute part of an experiment submission. The submission is only complete when experimental meta-data is also submitted to ArrayExpress using either MAGE-TAB or Annotare submission tools. See submissions help page for further details.

 

Transferring files with Windows Explorer (drag and drop)

Make sure you've opened a Windows Explorer window and not an internet browser window as the latter is not designed for FTP file uploads.

In the Windows Explorer window, enter ftp://ftp-private.ebi.ac.uk/ in the address bar. You will be asked to login. The username is aexpress and password is aexpress1:

After logging in, you can drag files from your computer's file system and drop them onto the logged-on Windows Explorer window.

Note: you will not be able to see any files or directories on the FTP site, including the files that you have uploaded.

 

Transferring files with Mac/Unix/Windows terminal (command prompt)

Connect to the FTP server using the command ftp ftp-private.ebi.ac.uk. The username is aexpress and password is aexpress1. Use the put command to place one file (or mput for multiple files) into the default FTP directory.

To exit FTP, type quit. On exiting you will get a message printed to screen to tell you whether your transfer was successful.

Note: you will not be able to list the files in the directory or download files from the FTP site to your directory.
